# 🍔 Online Food Ordering System

A simple **Online Food Ordering System** built using **Java (OOP concepts)** with a basic **HTML, CSS, JavaScript frontend**.
This project demonstrates how users can browse food items, add them to a cart, and place an order.

---

## 📌 Features

* 🧾 Display food menu
* ➕ Add items to cart
* 🛒 View cart with total price
* ✅ Place order functionality
* 💻 Simple and interactive UI

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Java (Object-Oriented Programming)
* **Concepts Used:** Encapsulation, Classes, Objects, Lists

## ▶️ How to Run the Project

### 🔹 Frontend (UI)

1. Open the project folder
2. Double-click on `index.html`
3. The application will open in your browser

---

### 🔹 Java Backend (Optional)

> Note: Java is used for demonstrating OOP concepts

1. Install JDK (Java 21 recommended)
2. Open terminal in project folder
3. Compile:

   ```
   javac *.java
   ```
4. Run:

   ```
   java Main
   ```
